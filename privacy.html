개인정보 처리방침

0. 개요

본 개인정보 처리방침은 2025년 3월 14일부터 적용됩니다.

Two Hearts(이하 "서비스")는 대한민국 「개인정보 보호법」 제30조에 따라 이용자의 개인정보를 보호하고, 관련 법령을 준수하고 있습니다.

1. 수집하는 개인정보

  • 검색어 자동완성 기능 사용 시
    • 검색어
    • 사용자의 IP 주소
    • 브라우저 및 운영체제 정보

2. 개인정보의 이용 목적

  • 검색어 자동완성 기능 제공
  • 서비스의 보안 유지 및 향상

3. 개인정보의 보유 및 파기

서비스는 이용자의 개인정보를 일주일 이내에 파기합니다.

4. 개인정보의 제3자 제공

4.1. 배경 이미지

이 서비스는 배경 이미지를 제공하기 위해 사용자의 브라우저가 특정 외부 서버에 요청을 보낼 수 있습니다.

4.2. 검색 엔진

서비스는 사용자의 검색어를 저장하지 않으며, 입력된 검색어는 사용자가 설정한 기본 검색 엔진으로 직접 전송됩니다.

Privacy Policy

0. Overview

This Privacy Policy has been in effect since March 14, 2025.

Two Hearts (the "Service") complies with South Korea's Personal Information Protection Act to protect users' personal information and adhere to relevant regulations.

1. Collected Personal Information

  • When using search autocomplete
    • Search queries
    • User's IP address
    • Browser and operating system information

2. Purpose of Personal Information Usage

  • Providing search autocomplete functionality
  • Maintaining and enhancing service security

3. Retention and Deletion of Personal Information

The service deletes collected personal information within one week.

4. Third-Party Sharing of Personal Information

4.1. Background Images

The service may request background images from external servers.

4.2. Search Engine

The service does not store user search queries. All queries are redirected to the search engine set by the user.
